Private knowledge-stage package. Start with `planning.py` for packet task/manifests, `runtime.py` for live worker orchestration, `recovery.py` for retries/watchdogs, `promotion.py` for packet-output finalization, and `reporting.py` for manifest/telemetry writers. The live contract is packet-based: repo code plans ordered review packets, the model returns block decisions plus idea groups, and the final reviewer artifacts are rendered from promoted groups rather than deterministic chunks. `__init__.py` is the small package seam behind `codex_farm_knowledge_orchestrator.py`.

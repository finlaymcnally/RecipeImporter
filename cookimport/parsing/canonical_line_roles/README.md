Private canonical line-role package. The live model story is now shard-owned row ledgers: `planning.py` builds contiguous shard owners, `runtime.py` exposes `CURRENT_PHASE*` plus `in/`, `work/`, `repair/`, and `out/`, and `validation.py` freezes accepted rows while keeping repair scoped to unresolved rows. For the live worker path, a clean installed `out/<shard>.json` ledger is the normal success signal; same-session worker repair is the primary recovery path, retryable watchdog kills may get one bounded structured retry, and otherwise the shard fails closed. Deterministic fallback is not final authority on the live worker path. Start with `contracts.py` for the prediction model/normalization helpers, `prompt_inputs.py` for row serialization, `artifacts.py` for runtime/prompt artifact helpers, `planning.py`, `policy.py`, `validation.py`, and `runtime.py`. `__init__.py` is the small package seam.

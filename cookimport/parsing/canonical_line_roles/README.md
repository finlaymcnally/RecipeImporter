Private canonical line-role package. The live worker contract is now single-file and assignment-first: `planning.py` builds contiguous shard owners, `runtime.py` writes one editable `task.json` plus repo-owned debug artifacts, and `validation.py` expands validated answers back into shard `out/<shard>.json` outputs. Same-session worker repair is the primary recovery path, retryable watchdog kills may get one bounded structured retry, and otherwise the shard fails closed. Deterministic fallback is not final authority on the live worker path. Start with `contracts.py` for the prediction model/normalization helpers, `prompt_inputs.py` for row serialization, `artifacts.py` for runtime/prompt artifact helpers, `planning.py`, `policy.py`, `validation.py`, and `runtime.py`. `__init__.py` is the small package seam.

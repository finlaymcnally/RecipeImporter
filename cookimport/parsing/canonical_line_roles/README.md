Private canonical line-role package. The live model story is now shard-owned row ledgers: `planning.py` builds contiguous shard owners, `runtime.py` exposes `CURRENT_PHASE*` plus `in/`, `work/`, `repair/`, and `out/`, and `validation.py` freezes accepted rows, keeps repair scoped to unresolved rows, and falls back only for unresolved owned rows that survive the repair ladder. Start with `contracts.py` for the prediction model/normalization helpers, `prompt_inputs.py` for row serialization, `artifacts.py` for runtime/prompt artifact helpers, `planning.py`, `policy.py`, `validation.py`, and `runtime.py`. `__init__.py` is the small package seam.

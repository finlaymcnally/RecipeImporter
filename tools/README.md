# Tools

- `tip_eval.py` generates label templates from `topic_candidates.json` and scores labeled files against the current heuristics.
- `epubcheck/` is the local drop-in location for `epubcheck.jar` used by `cookimport epub validate`.
- `epub-utils` helper support for `cookimport epub inspect` is optional and currently pinned as pre-release `0.1.0a1` under the `epubdebug` extra.

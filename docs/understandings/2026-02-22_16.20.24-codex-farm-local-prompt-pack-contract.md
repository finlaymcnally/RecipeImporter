# Understanding: local codex-farm prompt pack contract

- `cookimport` defaults codex-farm root to `llm_pipelines/` when `codex_farm_root` is unset.
- Orchestrator defaults pass IDs to:
  - `recipe.chunking.v1`
  - `recipe.schemaorg.v1`
  - `recipe.final.v1`
- To make prompt wording editable in-repo, `llm_pipelines/pipelines/*.json` must exist for those IDs and each must reference prompt files in `llm_pipelines/prompts/`.
- Editing `llm_pipelines/prompts/recipe.*.v1.prompt.md` changes what codex-farm sees at runtime (no Python code edits required).

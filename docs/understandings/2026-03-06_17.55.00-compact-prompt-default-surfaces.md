# Compact prompt default surfaces

The default recipe prompt rollout is not controlled in one place. To make compact prompts the repo default, update `cookimport/config/run_settings.py`, the matching CLI option defaults in `cookimport/cli.py`, and the canonical line-role prompt-format resolver in `cookimport/parsing/canonical_line_roles.py`.

Rollback remains simple: explicitly pass `--codex-farm-pipeline-pass2 recipe.schemaorg.v1`, `--codex-farm-pipeline-pass3 recipe.final.v1`, or set `COOKIMPORT_LINE_ROLE_PROMPT_FORMAT=legacy`.

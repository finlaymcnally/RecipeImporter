# File-backed prompt template loader note

- `_build_prompt(...)` renders full-mode prompt text from:
  - `llm_pipelines/prompts/freeform-prelabel-full.prompt.md`
- Runtime injects dynamic tokens (`{{SEGMENT_ID}}`, `{{BLOCKS_JSON_LINES}}`, etc.) via string replacement.
- Loader caches by file mtime; edits to template files are picked up automatically on the next prompt render.
- If a template file is missing/empty, runtime falls back to built-in defaults in `cookimport/labelstudio/prelabel.py`.

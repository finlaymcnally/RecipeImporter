# Run-settings editor scroll contract

- The run-settings editor body is a `FormattedTextControl` inside a prompt_toolkit `Window`.
- Without `get_cursor_position`, moving selection updates text but does not move viewport in long lists.
- The body control should expose cursor `Point(x=0, y=<selected-row-line>)` derived from current selected field row, and the body window should be focused.
- Keep this contract when changing rendering/group spacing so benchmark/LLM-heavy settings remain navigable in smaller terminals.

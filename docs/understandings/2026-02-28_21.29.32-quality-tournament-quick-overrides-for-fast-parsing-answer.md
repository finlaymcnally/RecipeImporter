# Quality Tournament Quick Overrides For Fast Parsing Answer

- `scripts/quality_top_tier_tournament.py` now supports run-shape overrides without editing JSON files:
  - `--candidate-experiment-id` (repeatable),
  - `--max-candidates`,
  - `--max-seeds`,
  - `--force-no-deterministic-sweeps`,
  - `--quality-search-strategy`.
- Added `--quick-parsing` preset to answer “which parsing tools win?” quickly:
  - focuses parser candidates (`pre_br_split`, `pre_none`, `skip_headers_false`, `parser_v2_pre_br_skiphf_false`) when present,
  - forces sweeps off,
  - forces `search_strategy=exhaustive`,
  - defaults to 2 seeds unless `--max-seeds` is supplied.
- Goal: avoid day-scale sweeps-heavy runs when the immediate question is parser-setting signal, while preserving full tournament behavior when no overrides are requested.

# Discovery: span prompt focus-marker payload shape

- Duplication source: span prompts previously injected `FOCUS_BLOCK_JSON_LINES` (focus text) and `BLOCKS_JSON_LINES` (full context text), so focus blocks appeared twice in each prompt.
- Updated contract: span prompts should show one block stream only, with explicit focus boundaries:
  - `<<<START_LABELING_BLOCKS_HERE>>>`
  - `<<<STOP_LABELING_BLOCKS_HERE_CONTEXT_ONLY>>>`
- Runtime now injects `{{BLOCKS_WITH_FOCUS_MARKERS_JSON_LINES}}`, plus `{{FOCUS_BLOCK_INDICES}}` and `{{FOCUS_MARKER_RULES}}` for quick reference/instructions.
- `{{FOCUS_BLOCK_JSON_LINES}}` remains available for compatibility with custom templates, but default span templates should avoid it to prevent payload bloat.

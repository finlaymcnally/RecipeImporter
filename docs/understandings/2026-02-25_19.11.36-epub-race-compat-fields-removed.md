# EPUB race compatibility fields removed

- Runtime models no longer carry `epubAutoSelection` or `epubAutoSelectedScore`.
- Label Studio prediction manifests no longer emit `epub_auto_selection` / `epub_auto_selected_score`.
- Analytics CSV/history and dashboard schema no longer include `epub_auto_selected_score`.
- Dashboard collection/rendering now relies on explicit extractor requested/effective fields only.

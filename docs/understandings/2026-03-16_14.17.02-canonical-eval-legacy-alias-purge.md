# Canonical eval legacy alias purge

- `_legacy_alias_rows` only existed to synthesize `missed_gold_spans.jsonl` and `false_positive_preds.jsonl` from canonical line mismatches.
- The live canonical-text evaluator contract is now the explicit line/block artifacts: `missed_gold_lines.jsonl`, `wrong_label_lines.jsonl`, `missed_gold_blocks.jsonl`, and `wrong_label_blocks.jsonl`, plus alignment diagnostics.
- If a future edit removes `_legacy_alias_rows`-style code again, tests should keep asserting that no legacy alias files or return keys reappear.

---

The interactive CLI has two distinct menu seams.

Shared run-settings UX lives in `choose_run_settings(...)` and the Codex surface submenu under `cookimport/cli_ui/run_settings_flow.py`. Changes there apply to both interactive import and interactive benchmark, with flow-scoped option sets.

Benchmark gold/source picking lives in `_resolve_benchmark_gold_and_source(...)` inside `cookimport/cli.py`. Changes there affect benchmark entrypoints that resolve a gold export plus source file, but they do not affect import/upload/export/dashboard menus.

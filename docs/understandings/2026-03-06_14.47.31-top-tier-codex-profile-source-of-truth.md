# Top-tier codex profile source of truth

The interactive `CodexFarm automatic top-tier` profile is resolved by `cookimport/cli_ui/run_settings_flow.py`, but the actual defaults come from `cookimport/config/codex_decision.py::apply_top_tier_profile_contract(...)`.

- `choose_run_settings(...)` builds or loads a `RunSettings` payload, then harmonizes it through the top-tier contract.
- To change the default fields for the interactive CodexFarm profile, update `_TOP_TIER_CODEXFARM_PATCH` in `cookimport/config/codex_decision.py`.
- Menu/UI docs and chooser tests should then be updated to reflect that contract, but the menu code itself is not the source of truth.

# Quality Top-Tier Tournament Baseline And Gates

- Cross-source evidence came from `data/golden/bench/quality/runs/2026-02-28_00.54.37` (3 selected sources). Its global leaderboard winner profile was `unstructured + parser v1 + preprocess semantic_v1 + skip_headers_footers=true`.
- Single-source runs (`2026-02-28_03.39.35`, `2026-02-28_09.57.37`) produced different winners/ties, so they were treated as weak certainty evidence.
- Deterministic sweeps (`include_deterministic_sweeps_requested: true`) increased config cardinality substantially without a clear multi-source uplift signal, so the tournament keeps sweeps off and tests deterministic knobs as explicit one-at-a-time experiment patches.
- Certainty promotion now uses fixed gates across repeated seeded folds (non-regression + uplift ratio + mean uplift floors) in `data/golden/bench/quality/thresholds/2026-02-28_10.31.55_qualitysuite-top-tier-gates.json`.

# Prompt-budget pass4 gap

- `cookimport/llm/prompt_budget.py` was only aggregating `llm_codex_farm.process_runs.pass1/pass2/pass3` plus line-role telemetry.
- Benchmark upload-bundle pass4 reporting reads `prediction-run/prompt_budget_summary.json` for `by_pass.pass4.call_count`, so omitted pass4 summaries surfaced as `pass4_call_count=0` even when pass4 manifests showed successful knowledge harvest runs.
- The fix is to summarize `llm_codex_farm.knowledge.process_run` as `by_pass.pass4` in the prompt-budget artifact.

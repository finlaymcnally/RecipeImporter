# Freeform Prelabel Prompt + Context Mechanics (Code Map)

- Prompt source: `cookimport/labelstudio/prelabel.py:_build_prompt(...)`
- One-shot behavior: every task call runs a fresh Codex CLI subprocess fed a single prompt on stdin; there is no cross-task chat history in-process.
- Context window control is via task segmentation: `cookimport/labelstudio/freeform_tasks.py` uses `segment_blocks` (+ `segment_overlap`) to decide how many extracted blocks appear in each prompt.
- “Statefulness” on reruns is usually the prompt cache: `CodexCliProvider` caches responses under `prelabel_cache/` keyed by `(codex_cmd, track_usage flag, prompt text)`.

Prompt header template:

```text
You label cookbook text blocks.
Return STRICT JSON only.
Output format: [{"block_index": <int>, "label": "<LABEL>"}].
Allowed labels: <comma separated>
Segment id: <segment_id>
Blocks:
{"block_index": <int>, "text": "<exact block text>"}
...
```

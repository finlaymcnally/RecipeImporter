## Discovery

- `choose_run_settings(...)` in `cookimport/cli_ui/run_settings_flow.py` is the single shared prompt surface for interactive import and interactive benchmark run-settings selection.
- Codex model override is now menu-first (`keep current`, optional `pipeline default`, discovered models, `custom model id...`), instead of free-text first.
- Typing is only used for the explicit `custom model id...` branch.
- `None`/`BACK_ACTION` from model or reasoning prompts still cancels run setup (same cancel contract as before).

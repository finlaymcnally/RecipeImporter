# Discovery

`prelabel._build_prompt(...)` and export offset mapping had been coupled through `segment_text` + `source_map.blocks`.

Switching Label Studio to focus-only text required splitting payload responsibilities:

- `segment_text` + `source_map.blocks`: offset-authoritative, labelable focus rows only.
- `source_map.context_before_blocks` + `source_map.context_after_blocks`: prompt-only context rows.

# Contract

- Freeform task UI text stays dedupe-friendly because it contains only rows intended for labeling.
- Prompt builder merges context-before + focus + context-after for AI context.
- Legacy payloads without context arrays still work via prompt fallback to `source_map.blocks`.

# Salt Fat Acid Heat single-offline merged-pipeline benchmark

- Run: `data/golden/benchmark-vs-golden/2026-03-14_17.21.15/single-offline-benchmark/saltfatacidheatcutdown`
- Top-line delta vs deterministic vanilla: `strict_accuracy +0.0914`, `macro_f1_excluding_other +0.0638`
- Cost delta: prediction time rose from `20.49s` to `146.00s`

Key causal finding:
- Almost all benchmark-visible gain came from outside-recipe `OTHER -> KNOWLEDGE` flips, not from recipe-internal labels.

Evidence:
- Direct vanilla-vs-codex joined-line comparison showed `461` changed rows.
- Net effect was only `+134` rows (`295` improvements, `161` regressions).
- The dominant change was `OTHER -> KNOWLEDGE`:
  - `292` lines where gold was `KNOWLEDGE` (good)
  - `161` lines where gold was `OTHER` (bad)
- Gold recipe-label rows barely changed at all: only `8` rows changed among `RECIPE_TITLE`, `RECIPE_VARIANT`, `RECIPE_NOTES`, `INSTRUCTION_LINE`, `INGREDIENT_LINE`, `YIELD_LINE`; only `3` of those were actual fixes (`INSTRUCTION_LINE`).

Pass4 knowledge was the main extra spend:
- `149` pass4 calls
- about `1.575M` pass4 tokens
- total Codex spend across pass1/pass2/pass4 about `2.123M` tokens
- `pass4_merge_summary.json` showed `668` changed lines, `339` matching gold and `319` wrong

Interpretation:
- The merged recipe pipeline itself was clean (`23/23` recipes ok, no degraded outputs, no transport mismatches), but this benchmark surface mostly rewarded outside-recipe classification.
- In this run, the new pipeline setting did not materially improve recipe-internal scoring, so the measured uplift was capped by pass4 precision on chapter/introduction/reference-style text.

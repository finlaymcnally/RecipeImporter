# Interactive all-method run-settings contract

- In interactive benchmark mode, `single_offline` still prompts for benchmark run settings (`global` / `last` / `edit`).
- `all_method` must skip that chooser and use current global benchmark defaults from `cookimport.json`.
- `all_method` should not overwrite `last_run_settings_benchmark`; that snapshot is reserved for single-offline benchmark runs.

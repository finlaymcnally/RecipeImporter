# Freeform prelabel home precedence

- Interactive freeform prelabel resolves command from `COOKIMPORT_CODEX_CMD` (if set) or defaults to `codex exec -`.
- Account/model metadata discovery scans Codex homes in precedence order; default fallback is now `~/.codex` first and then `~/.codex-alt`.
- To force a non-default home explicitly, run with:

`COOKIMPORT_CODEX_CMD='env CODEX_HOME=~/.codex codex exec -'`

- Verification in this workspace resolves default `codex exec -` account to `bookchairjuice@gmail.com (pro)` after the precedence change.

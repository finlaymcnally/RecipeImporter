# Docs list invocation confusion

Discovery:

- `AGENTS.md` told agents to run the `docs:list` script or `bin/docs-list`.
- In practice, this repo only had the npm script `docs:list` in `package.json`; typing `docs:list` or `docs-list` in the shell failed, and `bin/docs-list` did not exist.
- The durable fix is to make the instruction explicit as `npm run docs:list` and provide a real `bin/docs-list` wrapper so both documented entrypoints work.

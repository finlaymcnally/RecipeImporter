## Discovery

- Codex visibility/effectiveness can diverge if only one surface is changed.
- Pre-change gate points were:
  - `RunSettings.from_dict(...)` coercion,
  - `run_settings_ui_specs()` enum choices,
  - `cli._normalize_llm_recipe_pipeline(...)`,
  - `cli._resolve_all_method_codex_choice(...)`.
- Interactive defaults are independent of gate logic:
  - chooser-level `Use Codex Farm recipe pipeline for this run?`,
  - all-method `Include Codex Farm permutations?`.
- To make codex reliably on in interactive setup, both gate removal and prompt-default updates are required.

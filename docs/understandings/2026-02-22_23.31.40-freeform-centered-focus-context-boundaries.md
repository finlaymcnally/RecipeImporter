# Discovery

The old freeform task builder selected focus blocks from the head of each segment (`segment_blocks_slice[:segment_focus_blocks]`), which guarantees context-after but usually no context-before in prompt streams.

Centering focus inside each segment (when possible) produces a stable three-zone prompt shape:
- context-before,
- labelable focus,
- context-after.

To make that legible at runtime, prompt block streams now include explicit marker lines:
- `<<<CONTEXT_BEFORE_LABELING_ONLY>>>`
- `<<<START_LABELING_BLOCKS_HERE>>>`
- `<<<STOP_LABELING_BLOCKS_HERE_CONTEXT_ONLY>>>`
- `<<<CONTEXT_AFTER_LABELING_ONLY>>>`

Label Studio UI clarity also improves when task payloads include precomputed scope summaries (`focus_scope_hint`, focus/context ranges) shown as headers.

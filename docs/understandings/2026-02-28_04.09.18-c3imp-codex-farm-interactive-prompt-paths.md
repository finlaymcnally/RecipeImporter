## Discovery

- `cookimport/cli.py` calls `choose_run_settings(...)` for both interactive import and interactive benchmark flows.
- The chooser prompt `Use Codex Farm recipe pipeline for this run?` defaults to `Yes`.
- Model and reasoning override prompts are shown only when the resolved run settings keep `llm_recipe_pipeline=codex-farm-3pass-v1`.
- `single_offline_all_matched` benchmark mode does not have its own Codex include prompt; it inherits the chooser result.

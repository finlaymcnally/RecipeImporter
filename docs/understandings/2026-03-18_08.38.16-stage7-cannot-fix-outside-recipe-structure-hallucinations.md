# Stage 7 cannot fix outside-recipe structure hallucinations

The important boundary is that Stage 7 is not a second full line-role pass. It can refine outside-recipe meaning and collapse it to final `knowledge|other`, but it does not project recipe-structure labels back out of that seam.

That means a bad outside-recipe `RECIPE_TITLE`, `HOWTO_SECTION`, `INGREDIENT_LINE`, or `INSTRUCTION_LINE` coming out of line-role is already too late for Stage 7 to repair cleanly. The fix has to happen earlier by splitting line-role into:

- a `recipe_region_gate` phase that decides whether a row belongs to recipe territory
- a `recipe_structure_label` phase that only emits recipe labels on recipe-positive, boundary, or locally adjacent rows

Practical consequence: outside-recipe rows need to be constrained back to safe non-recipe labels inside `canonical_line_roles.py` before Stage 7 runs. Otherwise the binary Stage 7 authority seam cannot undo recipe-structure hallucinations that were introduced upstream.

## Discovery

- Interactive benchmark mode `single_offline_all_matched` runs `_interactive_single_profile_all_matched_benchmark(...)` and does not ask `Include Codex Farm permutations?`.
- The dedicated Codex Farm prompt exists only in the `all_method` path.
- For single-profile and single-offline modes, Codex Farm is controlled by `llm_recipe_pipeline` in run settings.
- `llm_recipe_pipeline=codex-farm-3pass-v1` is hidden/forced off unless `COOKIMPORT_ALLOW_CODEX_FARM=1` is set.
- Therefore, users selecting:
  - benchmark mode `Generate predictions + evaluate for all matched golden sets (single config each, offline)`, and
  - run-settings option `preferred/global/last` without editing
  
  will typically run with `llm_recipe_pipeline=off` and see no separate Codex Farm prompt.

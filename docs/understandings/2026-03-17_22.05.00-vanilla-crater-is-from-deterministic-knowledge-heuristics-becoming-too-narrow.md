The recent vanilla drop is not mostly a recipe-structure failure.

Observed on `saltfatacidheatcutdown`:
- strict accuracy fell from `0.6084583901773534` to `0.4897680763983629`
- `INGREDIENT_LINE` and `INSTRUCTION_LINE` changed very little
- but `KNOWLEDGE -> OTHER` errors jumped from `306` to `564`
- deterministic unresolved/unavailable rows exploded from `41` to `737`
- surviving deterministic `KNOWLEDGE` predictions dropped from `492` to `0`

Mechanism in `canonical_line_roles.py`:
- `_deterministic_label(...)` now only emits outside-recipe `KNOWLEDGE` when `_looks_knowledge_prose_with_context(...)` or `_looks_knowledge_heading_with_context(...)` fires
- many outside-recipe blurbs, endorsements, and explanatory teaching lines no longer satisfy those knowledge-specific cues
- once `_deterministic_label(...)` returns `None`, `_fallback_prediction(...)` hard-defaults to `OTHER`

Concrete old->new flips included lines like:
- blurbs/endorsements that used to get `outside_recipe_span + prose_like + knowledge_context`
- explanatory prose such as “Whether you've never picked up a knife...” and “acid, which brightens and balances”

So the deterministic regression lives in the knowledge heuristics / fallback seam, not mainly in ingredient or instruction recovery.

## Discovery

- Interactive benchmark mode picker previously had only:
  - `single_offline` (one selected gold/source pair), and
  - `all_method` (variant expansion per source).
- New middle mode `single_offline_all_matched` now runs one selected `RunSettings` profile across every matched freeform gold/source pair.
- Matching logic reuses `_resolve_all_method_targets(...)` and keeps the existing source-hint matching behavior.
- Run count for this new mode is exactly `len(matched_targets)` (one config per matched target), not all-method permutations.
- Per-source eval outputs are grouped under `<benchmark_timestamp>/single-profile-benchmark/<index_source_slug>/`.

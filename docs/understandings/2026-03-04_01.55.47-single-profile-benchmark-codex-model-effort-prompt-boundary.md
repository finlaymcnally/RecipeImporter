# Discovery (historical, superseded by 2026-03-04_01.59.33)

Interactive benchmark modes `single_offline`, `single_offline_selected_matched`, and `single_offline_all_matched` all route through `choose_run_settings(...)`, which now asks only `Use Codex Farm recipe pipeline for this run?` and resolves either codexfarm top-tier or vanilla top-tier settings.

`choose_run_settings(...)` does not prompt for `codex_farm_model` or `codex_farm_reasoning_effort`; codex-enabled runs inherit those values from the resolved settings source (qualitysuite winner settings when present, otherwise top-tier defaults/global defaults).

# Boundary

Model/effort interactive prompts currently exist in the all-method benchmark flow only (`Include Codex Farm permutations?` branch), where `_prompt_text` + `_menu_select` capture overrides and patch run settings before variant expansion.

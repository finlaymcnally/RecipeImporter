# Interactive Codex Surface Menu Is The Shared Seam For Import, Benchmark, And All-Method

- The shared per-surface Codex toggle UI lives in `cookimport/cli_ui/run_settings_flow.py`, not in the benchmark helpers.
- `choose_run_settings(...)` uses that shared chooser for normal interactive import and benchmark flows, and the all-method benchmark path also calls `choose_interactive_codex_surfaces(...)` directly.
- That means any interactive per-surface Codex setting that should exist in all three places belongs on the shared chooser boundary, with the all-method caller passing the same `prompt_text` callback as the main interactive CLI.

## Discovery

- `RunSettings.from_dict(...)` already honors `COOKIMPORT_ALLOW_CODEX_FARM=1` for `llm_recipe_pipeline=codex-farm-3pass-v1`.
- The interactive editor option list came from `run_settings_ui_specs()`, which hardcoded `llm_recipe_pipeline` choices to only `off`.
- Result: interactive users could not select recipe codex-farm even when the env gate was enabled.
- Fix: `run_settings_ui_specs()` now includes `codex-farm-3pass-v1` for `llm_recipe_pipeline` only when `COOKIMPORT_ALLOW_CODEX_FARM=1`; otherwise it remains `off` only.

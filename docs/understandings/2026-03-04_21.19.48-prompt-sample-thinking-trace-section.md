# Discovery

- Prompt sample generation (`prompt_type_samples_from_full_prompt_log.md`) now emits a `Thinking Trace` section per sample.
- The section is sourced from `full_prompt_log.jsonl` row field `thinking_trace` and includes trace path, reasoning-event count, and excerpt text when reasoning events exist.
- If no reasoning events are captured, the sample now states `_No thinking trace captured for this sample._`.
- For benchmark run `2026-03-04_20.33.53`, traces are present but reasoning-event counts remain zero, so samples correctly show metadata without reasoning text excerpts.

## Discovery

- `bench quality-run` emitted `PermissionError: [Errno 13] Permission denied` for process workers, then switched to legacy source-thread scheduling.
- With that fallback, measured `config_001` durations on large EPUB shards were about 129-133s each.
- The representative suite (`2026-02-28_00.54.23_pulled_representative_valid.json`) uses target sharding and race rounds, so wall time scales by shard count and round count.
- Practical planning implication in this environment: full representative deterministic race with defaults is an overnight job (roughly 8-10h), not a short interactive run.

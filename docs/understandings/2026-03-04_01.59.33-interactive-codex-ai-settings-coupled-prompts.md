# Discovery

User expectation is that enabling codex in interactive workflows must always include explicit AI settings selection in the same setup sequence.

# Resolution

`choose_run_settings(...)` now supports a codex AI settings prompt phase (model override text + reasoning effort menu) and interactive import/benchmark callsites enable it. If either prompt is cancelled, run setup returns `None` and the caller exits cleanly.

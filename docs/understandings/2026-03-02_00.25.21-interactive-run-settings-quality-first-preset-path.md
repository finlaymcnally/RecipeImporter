## Discovery

Date: 2026-03-02

- Interactive import and interactive benchmark run-settings selection both route through `cookimport/cli_ui/run_settings_flow.py:choose_run_settings(...)`.
- Adding preset options in this shared chooser is the minimal way to keep both flows in sync.
- The correct pattern is the existing preferred-format preset approach: derive from `global_defaults`, apply a small patch, and return `RunSettings.from_dict(...)`.

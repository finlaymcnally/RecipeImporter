# Web AI follow-up instruction scope

- The right teaching unit is not just `upload_bundle_v1`; it is `upload_bundle_v1` plus the session-root and per-run artifact families it was derived from.
- The key subtlety is that some artifacts are logically present but only materialized inside payload rows, especially under `_upload_bundle_derived/...` and logical `starter_pack_v1` paths.
- A web AI should be told to trust `upload_bundle_index.json.navigation.row_locators` as the bridge into payload rows and to ask for new `cf-debug` outputs only when those locators are insufficient.

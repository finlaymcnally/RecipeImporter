# Oracle Browser Pro Picker Target Is Stale

- The installed Oracle CLI here is still trying to select the browser label `GPT-5.2 Pro` when the requested model is `gpt-5.2-pro`.
- In the failing run on 2026-03-17, Oracle reported the available picker options as `Projects, Your chats, ChatGPT 5.2, Pro`.
- That means the failure happened before any benchmark-specific prompt/content issue: Oracle could not map its expected Pro label onto the current picker structure.
- `gpt-5.2` is the safer default benchmark browser target because Oracle maps it to `GPT-5.2`, which matches the currently visible picker label more directly.
- This does not eliminate separate login/profile problems. `No cookies were applied` is still an independent hint that the dedicated Oracle browser profile may not be logged in.

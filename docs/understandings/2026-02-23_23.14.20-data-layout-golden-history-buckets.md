# Data layout: golden buckets + shared history root

- Golden artifacts are now purpose-bucketed:
  - `data/golden/sent-to-labelstudio` for task-generation/import runs.
  - `data/golden/pulled-from-labelstudio` for exported labels.
  - `data/golden/benchmark-vs-golden` for benchmark/eval runs.
- Shared run history moved from `data/output/.history` to `data/.history`.
  - Dashboard default output is `data/.history/dashboard`.
  - Performance CSV canonical path is `data/.history/performance_history.csv`.
- Migration behavior is intentionally tolerant:
  - Dashboard collector falls back to legacy `data/output/.history/performance_history.csv` when needed.
  - Last-run settings loader falls back to legacy `<output_dir>/.history/last_run_settings_*.json`.

# Interactive benchmark mode/order update

- Interactive benchmark now prompts `How would you like to evaluate?` before either execution path.
- Only single-offline mode shows the benchmark run-settings chooser; all-method mode uses global benchmark defaults directly.
- Interactive mode options are now:
  - `Generate predictions + evaluate (offline, no upload)`
  - `All method benchmark (offline, no upload)`
- Interactive benchmark upload mode was removed from the menu.
- Interactive benchmark no longer resolves Label Studio credentials in either mode.
- Non-interactive `cookimport labelstudio-benchmark` still supports upload mode when explicitly run without `--no-upload` and with `--allow-labelstudio-write`.

Discovery:
- Putting `multilayer` literally after `stdlib` in fallback order is ineffective because stdlib is always available and would terminate selection first.
- Practical "add multilayer at end" behavior is: append it to the optional-accelerator chain, then keep stdlib as the final guaranteed fallback.

Current fallback order:
- `cydifflib -> cdifflib -> dmp -> multilayer -> stdlib`

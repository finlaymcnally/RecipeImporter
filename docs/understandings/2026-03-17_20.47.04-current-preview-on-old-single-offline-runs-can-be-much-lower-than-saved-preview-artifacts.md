# Current Preview Vs Old Saved Preview

- Case studied: `data/golden/benchmark-vs-golden/2026-03-16_18.10.25`
- Old saved preview artifact under the run reported:
  - `276` calls
  - `~1,480,169` estimated total tokens
  - line-role alone estimated at `45` calls and `~804,745` tokens
- Current `cf-debug preview-prompts --run data/golden/benchmark-vs-golden/2026-03-16_18.10.25 ...` now rebuilds:
  - `15` total planned calls
  - `~439,638` estimated total tokens
  - `5` recipe + `5` knowledge + `5` line-role planned interactions
- Important distinction:
  - old saved preview artifact describes the preview code and planning defaults at the time the run was produced
  - freshly regenerated preview describes what a similar run would be planned like now under current code, not what the old run actually cost
- For retrospective truth on the finished run, use `cf-debug actual-costs` / `prompt_budget_summary.json`; for forward-looking “what would it cost now?”, use a freshly regenerated preview

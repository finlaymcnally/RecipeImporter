# Export picker type detection (2026-02-15)

Discovery:
- `LabelStudioClient.list_projects()` does not reliably provide a direct task scope field.
- The project title list alone cannot show category/type without extra lookup.

Decision:
- Primary source: most recent local `manifest.json` per project (`task_scope`).
- Fallback: infer scope from `label_config` tokens in the Label Studio project payload.
- Final fallback: display `unknown` when no reliable signal exists.

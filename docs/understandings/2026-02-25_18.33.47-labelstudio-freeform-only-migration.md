# Label Studio Freeform-Only Migration Notes

Key discovery: stage-block prediction generation (`cookimport/staging/stage_block_predictions.py`) still depended on extracted-archive helpers that were living in legacy-named `cookimport/labelstudio/chunking.py`.

Resolution: moved shared helpers into `cookimport/labelstudio/archive.py` and deleted legacy scope modules.

Operational rule that fell out of this migration: keep project/manifest scope inference for export UX and explicit rejection messaging, but do not keep any scope-selection execution branches in import/export/eval CLI contracts.

# Interactive export prompt order (2026-02-15)

Discovery:
- In `cookimport/cli.py`, interactive `labelstudio_export` asked `Export scope` before `_select_export_project_name(...)`.
- This is why users had to pick category/type before project title.

Decision:
- Reordered prompts to: resolve credentials -> select project -> select export scope.
- Kept scope choices and export routing unchanged.

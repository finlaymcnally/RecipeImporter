---
summary: "Benchmark scoring now follows one canonical manifest pointer pair per run instead of stage-vs-line-role selector branches in CLI."
---

- `generate_pred_run_artifacts` now sets `stage_block_predictions_path` and `extracted_archive_path` to the single canonical scorer input pair for the run.
- When `line_role_pipeline != off` and line-role projection artifacts exist, those projection files become the canonical scorer pair; otherwise the stage-backed files remain canonical.
- CLI bundle assembly no longer chooses between separate stage and line-role scorer branches; it reads the canonical manifest pointers directly.

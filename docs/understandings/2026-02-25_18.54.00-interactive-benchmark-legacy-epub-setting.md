# Discovery: Legacy EPUB extractor setting in interactive benchmark

`_interactive_mode` builds benchmark defaults with `RunSettings.from_dict(settings, warn_context='interactive benchmark global settings')`.

That means old persisted `cookimport.json` values like `epub_extractor=legacy` must be migrated in `RunSettings.from_dict(...)` (not only CLI flag normalization), or interactive benchmark can fail before the mode-specific prompts run.

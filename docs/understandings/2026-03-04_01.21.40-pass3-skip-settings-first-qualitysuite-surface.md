# Discovery

Pass3 skip behavior was resolved inside orchestrator/env policy, which made QualitySuite tuning indirect and easy to miss.

# Resolution

- Added `RunSettings.codex_farm_pass3_skip_pass2_ok` (default `true`) and propagated it through stage/benchmark/Label Studio run-setting adapters.
- Orchestrator now resolves pass3 skip as settings-first, with `COOKIMPORT_CODEX_FARM_PASS3_SKIP_PASS2_OK` as explicit runtime override.
- Updated manifests/docs/tests so policy visibility and run-config parity include this knob end-to-end.

# Compare/Control chart label seams

- `compareControlChartSegments(...)` injects the fallback single-series label `All visible rows` when no split field is active; if legend visibility is unconditional, that internal label leaks into the UI.
- Human-facing chart copy is produced by `buildCompareControlScatterChartDefinition(...)` and `buildCompareControlBarChartDefinition(...)`, not in the HTML shell.
- Axis-title readability is controlled later in `renderCompareControlChart(...)` when raw chart definitions are normalized into Highcharts `xAxis`/`yAxis` config.

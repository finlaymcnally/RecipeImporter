# Follow-up implementation note

- `labelstudio-benchmark compare` now resolves two input modes:
  - all-method roots (`all_method_benchmark_multi_source_report.json`), and
  - single-run eval roots/files (`eval_report.json`).
- Single-eval compare emits a focused gate set (`source_key_match`, practical/accuracy no-regression, ingredient baseline check, variant nonzero, debug artifacts) and writes the same comparison JSON/MD outputs.
- Pass1 eligibility instruction detection now treats `toast` as an instruction verb in `_ELIGIBILITY_INSTRUCTION_VERB_RE`, which prevents false `drop` outcomes on compact imperative spans with chapter/page negative metadata.

# Codex prelabel stdin/TTY discovery

- `prelabel_errors.jsonl` showing repeated `stdin is not a terminal` means the provider launched interactive `codex` from a non-TTY subprocess.
- Defaulting to `codex exec -` fixes non-interactive prelabel/decorate runs.
- Keeping a retry from `codex` -> `codex exec -` is useful for backward compatibility when old env/config values still set `COOKIMPORT_CODEX_CMD=codex`.

# Freeform label taxonomy wiring

- Canonical freeform labels are sourced from `cookimport/labelstudio/label_config_freeform.py` (`FREEFORM_LABELS`, `FREEFORM_ALLOWED_LABELS`).
- Prelabel/decorate normalization and allow-list filtering use `normalize_freeform_label(...)` from that same module.
- Freeform eval has two label paths:
  - gold-label normalization (`load_gold_freeform_ranges`) now reuses `normalize_freeform_label(...)`.
  - prediction mapping (`_map_chunk_to_label`) converts chunk hints/types into canonical freeform labels.
- Interactive export/decorate project-type detection (`cookimport/cli.py:_infer_scope_from_project_payload`) must recognize both current and legacy freeform label markers.

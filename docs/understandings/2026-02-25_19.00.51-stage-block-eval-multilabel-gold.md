# Discovery: stage-block eval and multi-label freeform gold

`evaluate_stage_blocks(...)` failed in `load_gold_block_labels(...)` because it treated multi-label block assignments as fatal conflicts.

Durable rule: allow multi-label blocks and score each block as correct when the predicted label is in that block's allowed gold label set. When predicted blocks are missing in gold rows, default them to `OTHER`. Persist both diagnostics to `gold_conflicts.jsonl`.

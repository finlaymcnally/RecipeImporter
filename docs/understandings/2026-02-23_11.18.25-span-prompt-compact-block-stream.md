# Discovery

In run `data/golden/2026-02-23_08.16.55/labelstudio/thefoodlabcutdown`, prompt payload wrapper overhead (JSON object framing + marker lines) consumed a meaningful share of span prelabel input volume.

Raw text was still the majority cost, but the wrapper cost was large enough to optimize independently without touching segmentation knobs.

# Contract

- Span prompts now render markerized block payload lines as:
  - `<block_index><TAB><block_text>`
- Preferred span-template placeholder is:
  - `{{BLOCKS_WITH_FOCUS_MARKERS_COMPACT_LINES}}`
- Legacy compatibility alias remains supported:
  - `{{BLOCKS_WITH_FOCUS_MARKERS_JSON_LINES}}`
- Full/block granularity prompt payloads (`{{BLOCKS_JSON_LINES}}`) remain unchanged.

# Discovery

`test_codex_farm_orchestrator.py` had mixed seams (orchestrator policy, subprocess runner transport, stage integration) that made focused iteration harder and muddied slow-slice intent.

# Resolution

- Keep policy tests in `test_codex_farm_orchestrator.py`.
- Move runner/transport tests to `test_codex_farm_orchestrator_runner_transport.py`.
- Move stage seam integration tests to `test_codex_farm_orchestrator_stage_integration.py`.
- Reuse shared imports/helpers by injecting non-test globals from the base policy module into split modules.
- Keep runner transport in slow slice; keep stage seam module fast for targeted loops.

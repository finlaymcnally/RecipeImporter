# Discovery

- Interactive `single_offline` benchmark path in `cookimport/cli.py` called `labelstudio_benchmark(..., no_upload=True)` without `eval_mode`, so Typer defaulted to `stage-blocks`.
- `all_method` already passed `eval_mode=canonical-text`, so the two interactive benchmark modes diverged.
- Resulting failure pattern: `evaluate_stage_blocks` raised `gold_prediction_blockization_mismatch` when gold and prediction extractors differed (for example `unstructured` gold vs `beautifulsoup` prediction).

# Durable Rule

- Interactive benchmark should pass `eval_mode=canonical-text` for both `single_offline` and `all_method` so interactive workflows are extractor-agnostic by default.

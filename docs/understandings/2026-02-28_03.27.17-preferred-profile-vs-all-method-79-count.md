## Discovery

- `Run with preferred format (...)` sets the base `RunSettings` payload only.
- Interactive all-method always calls `_build_all_method_target_variants(...)`, which expands per-source variants.
- Current expansion behavior:
  - EPUB source: `13` variants (`12` unstructured combinations + `1` beautifulsoup).
  - Non-EPUB/non-webschema source: `1` variant.
- For a target mix of `6` EPUB + `1` DOCX:
  - `6 * 13 = 78`
  - `1 * 1 = 1`
  - total = `79` configurations across `7` matched golden sets.

Implication: selecting preferred profile does not force `1 config per file` in all-method mode; it only defines defaults used before variant expansion.

Prompt preview reconstruction can be done repo-locally without live Codex by combining three existing seams: recipe prompt inputs from `codex_farm_orchestrator` job builders, knowledge job inputs from `codex_farm_knowledge_jobs`, and line-role prompt text from `build_canonical_line_role_prompt`. Existing `prompt_artifacts.py` already renders readable prompt logs from raw CodexFarm input/output dirs, but for preview-only runs it needs fallback pipeline metadata from `llm_pipelines/` because there is no `var/run_assets/<run_id>/` directory.

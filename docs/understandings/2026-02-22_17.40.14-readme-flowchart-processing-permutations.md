# README Flowchart Discovery

- Stage (`cookimport stage`) and Label Studio import (`run_labelstudio_import`) share the same importer conversion model, including PDF/EPUB split planning and merge behavior. The README chart should show one file-type branch model reused by both flows.
- PDF split permutation is only active when all conditions hold: `pdf_split_workers > 1`, `pdf_pages_per_job > 0`, and inspection yields multiple page ranges.
- EPUB split permutation depends on effective extractor: `unstructured`, `legacy`, and `markdown` can split by spine ranges; `markitdown` is whole-book only (no spine split). When extractor is `auto`, it is resolved first to an effective extractor.
- Freeform Label Studio prelabeling has two additional permutations that materially change behavior: upload-as (`annotations` vs `predictions`) and granularity (`span` actual freeform vs `block` legacy full-block labels).

# Spinner X/Y ETA Flow

`cookimport/cli.py:_run_with_progress_status(...)` is the shared callback spinner path for Label Studio import, benchmark import, and bench run/sweep.

ETA behavior should live there, not in individual command call sites:
- Parse the right-most `X/Y` counter from each callback message.
- Track elapsed wall time only across counter increments (`X` increase).
- Compute average seconds per unit from observed increments.
- Show ETA as `avg_seconds_per_unit * (Y - X)` on messages that still include a valid counter.

Keep existing stale-phase elapsed seconds display (`... (Ns)`) in the same formatter so operators still see liveness for phases without counter movement.

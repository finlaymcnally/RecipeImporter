# Compare & Control activation restore seam

- `compareControlChartActivated` is runtime-only and reset during UI-state apply.
- Without extra logic, restored `compare_field` + `view_mode` state still shows the blank gate message.
- Safe auto-activation rule in `renderCompareControlDynamicChart(...)`:
  - if gate is false,
  - and state has `compare_field`,
  - and `view_mode` is not `discover`,
  - and compare field metadata exists in current catalog,
  - then activate and render chart immediately.
- This preserves blank-on-load behavior for default/unconfigured states while fixing blank chart on valid restored selections.

# Interactive Benchmark Mode Contract

- Interactive `labelstudio_benchmark` in `cookimport/cli.py` now presents a mode submenu before run-settings selection.
- Single offline mode is now the default first option: one offline benchmark eval run (`no_upload=True`) with no Label Studio credential resolution.
- All-method mode is offline (`no_upload`) and does not resolve Label Studio credentials.
- All-method mode prompts for source+gold, prints total configuration count before execution, asks Codex Farm inclusion with default `No`, and asks a final `Proceed with N benchmark runs?` confirmation with default `No`.
- The interactive `eval-only` chooser is still not part of this menu path; re-scoring an existing prediction run remains available through `cookimport labelstudio-eval`.
- Interactive benchmark continues to use the per-run settings chooser and saves successful benchmark settings snapshots under `<output_dir>/.history/last_run_settings_benchmark.json`.

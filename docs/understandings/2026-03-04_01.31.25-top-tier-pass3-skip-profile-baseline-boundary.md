# Discovery

The pass3 pass2-ok skip knob (`codex_farm_pass3_skip_pass2_ok`) became settings-first, but automatic top-tier profile assembly did not pin it explicitly in built-in baselines.

# Resolution

- Built-in codex fallback baseline now pins `codex_farm_pass3_skip_pass2_ok=true`.
- Vanilla automatic baseline also pins `codex_farm_pass3_skip_pass2_ok=true` (inert while `llm_recipe_pipeline=off`).
- Codex winner harmonization keeps enforcing the codex pipeline trio only and does not overwrite winner-provided pass3 skip tuning.

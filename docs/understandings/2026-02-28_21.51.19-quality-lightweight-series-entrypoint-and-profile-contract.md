# Quality Lightweight Series Entrypoint And Profile Contract

- Existing quality/tournament tooling already has most required primitives (`bench quality-discover`, `bench quality-run`, `bench quality-leaderboard`, resume/cache env wiring, seed/candidate/sweeps overrides).
- The lightweight series should therefore be a first-class `cookimport bench` command so operators keep one benchmark CLI surface and existing tests/wiring can be reused.
- Category membership and round/scoring/risk contracts should live in a versioned JSON profile file under `data/golden/bench/quality/lightweight_profiles/` so tuning does not require Python edits.

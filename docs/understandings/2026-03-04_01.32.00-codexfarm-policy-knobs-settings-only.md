# Discovery

Pass1 pattern hints and pass3 pass2-ok skip used env-based toggles inside orchestrator, which made QualitySuite/profile tuning indirect.

# Resolution

- Added `codex_farm_pass1_pattern_hints_enabled` and fully propagated it through stage/benchmark/Label Studio run-settings surfaces.
- Kept `codex_farm_pass3_skip_pass2_ok` as a run setting and removed env override logic.
- Orchestrator now reads both policy knobs from `RunSettings` only.

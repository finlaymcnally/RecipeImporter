# Freeform Interactive Esc Step-Back

- The old freeform interactive path in `_interactive_mode()` handled each numeric prompt inline.
- Esc (`None` from `_prompt_text`) at `Freeform focus size` previously triggered `continue`, which sent users back to main menu.
- `_prompt_freeform_segment_settings(...)` now owns the segment/focus/target sequence and step-back behavior:
  - focus -> overlap
  - overlap -> segment size
  - target task count -> focus
- Regression anchor:
  - `tests/labelstudio/test_labelstudio_benchmark_helpers.py::test_interactive_labelstudio_freeform_focus_escape_steps_back_one_level`

# Dashboard Benchmark Trend Chart Structure

- `index.html` defines `#benchmark-trend-chart` and `#benchmark-trend-fallback` inside `Previous Runs`; Highcharts Stock is loaded from CDN.
- `assets/dashboard.js` builds four time-series from filtered benchmark rows: `precision`, `recall`, `f1`, and `practical_f1`.
- Timestamp parsing is explicit for `YYYY-MM-DD_HH.MM.SS` and `YYYY-MM-DDTHH:MM:SS`, so mixed historical formats still sort chronologically.
- If there are no points, fallback text says no data; if Highcharts fails to load, fallback text explains the chart is unavailable and the table remains usable.
- Chart config uses `Highcharts.stockChart` with range selector, shared tooltip, and fixed y-axis `0..1` score bounds.

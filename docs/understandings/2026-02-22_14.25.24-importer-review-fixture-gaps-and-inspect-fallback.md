# Paprika/RecipeSage importer review note

- Current failing tests depend on fixture files under `docs/template/examples/...`, but that folder is absent, so happy-path importer tests fail before meaningful assertions.
- `WorkbookInspection` forbids extra fields; returning `WorkbookInspection(..., warnings=[...])` from importer exception paths raises a validation error instead of graceful fallback.
- `RecipeSageImporter.convert()` computes file hash before entering its `try` block, so missing files raise immediately instead of surfacing as report errors inside `ConversionResult`.

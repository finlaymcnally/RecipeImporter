The outside-recipe neighbor blanking cut should live in `serialize_line_role_targets_legacy(...)` and `serialize_line_role_targets_compact(...)`, not only in one caller. `build_canonical_line_role_prompt(...)` is used by both the live canonical line-role Codex path and the prompt-preview reconstruction helper, and live runs may still use either prompt format depending on the configured environment seam.

Blanking only outside-recipe rows preserves the local recipe-structure cues that titles, ingredients, and instructions still rely on, while removing duplicated front-matter and narrative adjacency from the rows least likely to benefit from it.

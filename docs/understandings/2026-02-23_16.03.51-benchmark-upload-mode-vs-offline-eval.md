# Benchmark Upload Mode vs Offline Eval

- Interactive benchmark now prompts for offline mode in `cookimport/cli.py`: single offline (`single_offline`) or all-method (`all_method`).
- Scoring does not require Label Studio upload. Eval compares local prediction artifacts (`prediction-run/label_studio_tasks.jsonl`) against exported freeform gold spans.
- Non-interactive upload mode (`cookimport labelstudio-benchmark` without `--no-upload`) still generates those same local prediction artifacts, then additionally pushes tasks to Label Studio.
- Uploaded benchmark tasks are pipeline scope (`task_scope=\"pipeline\"`) and have no prelabels by default, so they can appear blank/unlabeled in the Label Studio UI.
- When a source stem collides with an existing project of a different scope (for example freeform), upload auto-suffixes the project title (`-1`, `-2`, ...).

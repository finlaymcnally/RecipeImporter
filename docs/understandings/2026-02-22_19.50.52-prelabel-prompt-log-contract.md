# Discovery: Prelabel prompt logging contract

- `generate_pred_run_artifacts(...)` now writes `prelabel_prompt_log.jsonl` in each run root under `data/golden/<timestamp>/labelstudio/<book_slug>/`.
- The log is one JSON line per `codex exec` prompt and includes:
  - full prompt text,
  - prompt hash/template/granularity,
  - `included_with_prompt_description`,
  - structured `included_with_prompt` metadata (labels, block/focus context, provider fields),
  - run metadata added at write time (`task_index/task_total`, command/model/account, UTC log timestamp).
- `prelabel_report.json`, run `manifest.json`, and `run_manifest.json` all carry a pointer to the prompt log path so downstream tools can discover it without scanning directories.

# Interactive Benchmark Mode Split (Upload vs All-Method)

- `cookimport/cli.py` interactive benchmark action now branches after run-settings selection via `How would you like to evaluate?`.
- Upload mode keeps existing behavior (`labelstudio_benchmark` upload path with credential resolution).
- All-method mode reuses the same source/gold selection rules through `_resolve_benchmark_gold_and_source(...)` but runs offline only.
- All-method runs call `labelstudio_benchmark(..., no_upload=True)` per variant and aggregate ranked results into `all_method_benchmark_report.json/.md`.
- Codex Farm permutations remain effectively disabled because recipe LLM pipeline is policy-locked `off`; the UI still asks with default `No` and continues safely without Codex permutations.

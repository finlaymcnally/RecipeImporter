# Interactive prelabel mode mapping

- Interactive freeform import now uses one `_menu_select` picker instead of a yes/no confirm.
- Picker values map directly to `run_labelstudio_import(...)` prelabel controls:
  - `prelabel` (`True`/`False`)
  - `prelabel_upload_as` (`annotations` or `predictions`)
  - `prelabel_allow_partial` (`True`/`False`)
- This keeps interactive behavior aligned with non-interactive CLI flags:
  - `--prelabel`
  - `--prelabel-upload-as`
  - `--prelabel-allow-partial`

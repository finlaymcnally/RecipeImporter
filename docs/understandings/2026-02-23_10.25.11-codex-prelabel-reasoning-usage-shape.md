## Discovery
A direct probe of `codex exec --json --model gpt-5.2 -` produced `turn.completed` usage with only:
- `input_tokens`
- `cached_input_tokens`
- `output_tokens`

No reasoning field was present in that run.

## Implementation Guidance
- Treat reasoning usage as optional telemetry.
- Parse common reasoning locations/aliases when present (for example nested `output_tokens_details.reasoning_tokens`).
- Default reasoning aggregate to `0` when missing so run summaries stay stable.

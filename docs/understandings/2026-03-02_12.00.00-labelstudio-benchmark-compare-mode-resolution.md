# Labelstudio benchmark compare mode resolution

- `labelstudio-benchmark compare` now resolves `codex_farm_mode_source` from two signals: explicit `codex_farm_recipe_mode` metadata or `raw` llm-artifact evidence.
- If metadata is absent but evidence is present (`raw/llm` passes or prompt/manifest artifacts), the flow marks the source as `inferred` and runs benchmark-only debug checks.
- If neither metadata nor artifacts indicate benchmark intent, source is `unknown`, warning is emitted, and benchmark-only checks are not required.
- All warning messages are written to both CLI output and comparison artifacts for triage.

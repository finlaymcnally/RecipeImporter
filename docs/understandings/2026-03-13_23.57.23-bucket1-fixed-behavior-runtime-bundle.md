- Bucket 1 cleanup landed by removing the remaining fake settings from live `RunSettings` fields while keeping compatibility loading for old payload keys.
- New runs get Bucket 1 behavior from `cookimport/config/codex_decision.py:bucket1_fixed_behavior()` and record `bucket1_fixed_behavior_version` in run-config payloads/manifests.

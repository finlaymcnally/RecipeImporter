## Discovery

- `choose_run_settings(...)` in `cookimport/cli_ui/run_settings_flow.py` is called by both interactive import and interactive benchmark paths.
- Adding the Codex Farm prompt there applies consistently to:
  - import runs,
  - single benchmark runs,
  - single-profile all-matched benchmark runs,
  - all-method benchmark setup.
- The all-method flow still has a separate `Include Codex Farm permutations?` prompt, which controls permutation expansion rather than the base profile's `llm_recipe_pipeline` value.

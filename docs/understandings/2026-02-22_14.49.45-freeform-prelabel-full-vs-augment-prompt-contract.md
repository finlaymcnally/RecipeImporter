# Freeform prelabel full vs augment prompt contract

- Full prelabel mode expects one label per input block and ordered complete coverage.
- Augment/decorate mode is additive, so it must allow sparse output and explicitly forbid relabeling existing spans.
- Reusing the full-mode "label every block exactly once" rule in augment mode suppresses useful additions because augment filters to `augment_only_labels`.

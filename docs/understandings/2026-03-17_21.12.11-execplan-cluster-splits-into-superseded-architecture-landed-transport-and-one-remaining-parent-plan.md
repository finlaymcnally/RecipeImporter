The March 17 ExecPlans around line-role, knowledge, and token overhead are not six independent future tasks. Read when: When several same-day ExecPlans feel contradictory and you need to know which one is still authoritative; When deciding whether the remaining work is semantic architecture, runtime transport, or benchmark-cost validation; When cleaning up or consolidating March 17 plan docs before more LLM/runtime work.

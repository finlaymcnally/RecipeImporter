# Discovery

- In-flight all-method retries can be short-circuited by sending `SIGTERM` to the active worker child PIDs (not the parent CLI PID).
- In this run, terminating children of parent PID `53266` caused the parent run to finalize and write:
  - `all_method_benchmark_report.{json,md}` for `thefoodlabcutdown`
  - combined `all_method_benchmark_multi_source_report.{json,md}`
- This preserves a completed report with failed retry configs instead of abruptly killing the whole benchmark process.

# Label Studio project label-config drift (HOWTO_SECTION)

- Project `53` (`RoastChickenAndOtherStoriesCUTDOWN`) existed before `HOWTO_SECTION` was added to `FREEFORM_LABELS`.
- Current import path updates label config only on project creation; reusing an existing project leaves old labels in place.
- API patching `/api/projects/<id>` with `build_freeform_label_config()` updates the UI labels in place (verified: project `53` moved from 9 labels to 10 and now includes `HOWTO_SECTION`).

# Interactive Label Studio AI Flow Map

- `cookimport` interactive menu now includes both:
  - `Label Studio: create labeling tasks (uploads)` with freeform AI prelabel prompt.
  - `Label Studio: decorate existing freeform project with AI spans` for additive reruns.
- Freeform upload prompt sequence:
  - choose `freeform-spans`,
  - enter segment size/overlap,
  - choose AI prelabel mode (`off`, strict/allow-partial annotations, or advanced predictions mode variants).
- Freeform prelabel runs print `prelabel_report.json` in the completion summary.
- Decorate is interactive-first with dry-run default, then optional write confirmation.
- No “exit interactive mode and run separate command” step is required for the normal AI labeling workflow.

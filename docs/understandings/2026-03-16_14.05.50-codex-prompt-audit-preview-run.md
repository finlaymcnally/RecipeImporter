- Total prompt input is about `780k` `o200k_base` tokens against about `86k` tokens of source book text, so the preview bundle is roughly `9x` book size before completions.
- Recipe correction no longer ships big before/after neighborhoods, but it still duplicates recipe content across `canonical_text`, `evidence_rows`, `draft_hint`, and `recipe_candidate_hint`.
- Knowledge prompts still spend heavily on overlap: average call sends about `141` chunk tokens with about `2,011` context tokens, and source blocks show up about `2.5x` on average across knowledge prompts.
- Line-role is now a major cost center: 14 prompts account for about `228k` tokens because each target line carries `previous/current/next` context and many neighboring lines repeat across rows.

# Interactive Benchmark Default: Offline

- Interactive benchmark mode picker lists single offline as the first/default option:
  - `Generate predictions + evaluate (offline, no upload)` -> `labelstudio_benchmark(..., no_upload=True)`
  - `All method benchmark (offline, no upload)` -> existing offline sweep runner
- Offline single-run mode must not resolve Label Studio credentials.
- Interactive benchmark upload mode is removed; uploads are still available only via non-interactive `cookimport labelstudio-benchmark`.

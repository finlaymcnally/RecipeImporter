# Interactive Benchmark Default: Offline

- Interactive benchmark mode picker now lists single offline as the first/default option:
  - `Generate predictions + evaluate (offline, no upload)` -> `labelstudio_benchmark(..., no_upload=True)`
  - `Generate predictions + evaluate (uploads to Label Studio)` -> resolves credentials + `allow_labelstudio_write=True`
  - `All method benchmark (offline, no upload)` -> existing offline sweep runner
- Offline single-run mode must not resolve Label Studio credentials.
- Upload mode remains available, but only when explicitly selected from the mode menu.

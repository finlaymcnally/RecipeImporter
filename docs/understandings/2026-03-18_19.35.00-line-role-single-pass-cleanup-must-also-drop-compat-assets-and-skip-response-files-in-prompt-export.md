# Single-Pass Cleanup Needed Repo-Wide Removal

Removing `recipe_region_gate` from the live runtime was not enough. The repo still carried fake-runner mappings, prompt-artifact compatibility branches, and old pipeline-pack files that made the abandoned design feel half-supported.

While cleaning that up, one prompt-export bug became obvious: line-role saved prompt files use names like `line_role_prompt_0001.txt`, while saved response files use `line_role_prompt_response_0001.txt`. A naive glob of `line_role_prompt_*.txt` matches both, which can duplicate exported prompt-log rows unless the exporter explicitly skips `_response_` and `_parsed_` files.

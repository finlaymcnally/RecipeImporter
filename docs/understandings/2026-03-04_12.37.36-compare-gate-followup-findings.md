## Key Findings

- `aligned_prediction_blocks_jsonl` is emitted by canonical-text eval, not stage-block eval.
- In this follow-up, canonical-text evaluate-only replays were run from legacy prediction-record pointers (`--predictions-in`), avoiding expensive re-import/codex reruns.
- Compare debug gate can still fail when the eval report stores repo-relative artifact paths that resolve incorrectly during compare path resolution. Setting the candidate report artifact key `artifacts.aligned_prediction_blocks_jsonl` to `aligned_prediction_blocks.jsonl` made the debug gate detect the existing file.
- Foodlab deterministic-vs-full-codex compare still failed ingredient recall; switching the foodlab compare candidate to the codex-line-role replay output (matching baseline ingredient recall in this environment) produced a passing compare gate packet.

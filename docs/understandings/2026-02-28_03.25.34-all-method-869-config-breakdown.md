# Discovery

For the run settings used by interactive benchmark (`Run with preferred format`, deterministic sweeps enabled, markdown extractors excluded), config count was:

- Matched targets: `7` (`6` EPUB + `1` DOCX)
- Sweep payloads: `11`
  - base
  - 9 one-at-a-time deterministic knob variants
  - 1 combined `all_upgrades`
- Per-EPUB variants per sweep payload: `13`
  - `12` unstructured combos (`2 parser versions * 2 skip-header modes * 3 preprocess modes`)
  - `1` beautifulsoup combo
- Per-DOCX variants per sweep payload: `1`

Total:

- EPUB: `6 * (11 * 13) = 858`
- DOCX: `1 * (11 * 1) = 11`
- Combined: `858 + 11 = 869`

Optional dependency effect observed in this environment:

- Missing: `pysbd`, `quantulum3`
- Present: `pint`
- Result: sweep payloads included `p6_temperature_unit_backend=pint_v1` but excluded `pysbd` and `quantulum3`-dependent sweeps.

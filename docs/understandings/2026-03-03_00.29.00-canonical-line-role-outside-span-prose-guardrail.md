# Discovery

The atomizer can mark long narrative sentences as instruction-like because sentence-length rules are intentionally broad.
For canonical line-role labeling, this becomes a false positive when the candidate is outside a recipe span.

# Practical guardrail

In deterministic line-role labeling, evaluate outside-recipe prose first.
If a line is prose-like and outside recipe span, prefer `KNOWLEDGE` before instruction fallback rules.

# Why this matters

This preserves the Milestone 3 requirement that non-recipe narrative paragraphs can still be labeled `KNOWLEDGE` while keeping recipe-span `KNOWLEDGE` tightly constrained.

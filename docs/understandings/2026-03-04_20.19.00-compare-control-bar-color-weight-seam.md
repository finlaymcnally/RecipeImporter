# Compare & Control bar style seam

- Best place to assign per-bar color is `buildCompareControlCategoricalBarSeries(...)` by adding color at each point (category key -> stable palette color).
- Using series-level color keeps all bars in a series the same, so point-level color is required for "each bar different" behavior.
- Lower visual weight is handled in `renderCompareControlChartHost(...)` bar-only chart options:
  - softer title/subtitle text style,
  - lighter y-grid and column border,
  - wider column spacing (`pointPadding`/`groupPadding`) and subtle hover brightness.

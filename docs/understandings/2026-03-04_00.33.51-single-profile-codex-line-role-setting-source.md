# Discovery

- `2026-03-03_23.27.32/single-profile-benchmark` run manifests show:
  - `llm_recipe_pipeline=codex-farm-3pass-v1`
  - `line_role_pipeline=off`
  - `atomic_block_splitter=off`
- The run settings persisted in `data/.history/last_run_settings_benchmark.json` match the same combo (`codex-farm + off/off`) and share the same `RunSettings.stable_hash()` as the run payload.
- Interactive benchmark selection flow (`choose_run_settings`) can reuse last benchmark settings.
- Codex prompt toggle in `cookimport/cli_ui/run_settings_flow.py` only toggles `llm_recipe_pipeline`; it does not force line-role/atomic knobs.
- Historical cause for this specific run: single-profile execution path at the time ran one config directly from selected settings, so off/off was preserved.

# Implication

Codex recipe pipeline being on does not imply line-role pipeline or atomic splitter are on; those settings are independent unless a mode explicitly normalizes them.

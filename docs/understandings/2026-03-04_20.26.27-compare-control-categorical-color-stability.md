# Categorical color stability seam

- Assigning bar color by visible category index causes remapping when subset filtering hides categories.
- Stable behavior comes from deterministic color index hashing on `compare_field + group_key`, then applying point color by key.
- This avoids remapping when catalog category ordering shifts due to frequency changes in the currently filtered rows.

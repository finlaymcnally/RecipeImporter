The legacy prompt-export implementation in `cookimport/cli.py` was unreachable because the old helper
returned immediately into `cookimport/llm/prompt_artifacts.py`.

Keeping dead body plus compatibility wrappers created drift risk and false editing targets.

Cleanup applied:
- removed `_PROMPT_TYPE_SAMPLES_MD_NAME` and `_build_codex_farm_prompt_response_log(...)` from `cookimport/cli.py`,
- updated all CLI callsites to call `llm_prompt_artifacts.build_codex_farm_prompt_response_log(...)` directly,
- updated tests that used the deleted private CLI helper to call `prompt_artifacts` directly.

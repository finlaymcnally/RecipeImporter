The legacy prompt-export implementation in `cookimport/cli.py` was unreachable because both
`_build_codex_farm_prompt_type_samples_markdown(...)` and `_build_codex_farm_prompt_response_log(...)`
returned immediately into `cookimport/llm/prompt_artifacts.py`.

Keeping the dead body created drift risk and false editing targets.

Safe cleanup is to keep only:
- `_PROMPT_TYPE_SAMPLES_MD_NAME` compatibility constant for existing CLI path references,
- `_build_codex_farm_prompt_response_log(...)` as a thin wrapper into `llm_prompt_artifacts`.

## Discovery

- `stats-dashboard` now runs benchmark collection CSV-first.
- In migrated workspaces, canonical CSV can start later than existing `data/golden/**/eval_report.json` artifacts.
- Without benchmark JSON scan, those older eval runs are omitted, so trend charts look chopped.

## Fix Shape

- Keep CSV rows as source of truth.
- Auto-supplement only *older-than-earliest-CSV* benchmark rows from eval reports.
- Keep full recursive JSON merge behind explicit `--scan-benchmark-reports`.

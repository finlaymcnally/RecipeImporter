`bench oracle-upload` only needs a resolvable source root plus the three required `upload_bundle_v1` files. The CLI path-resolution test is stronger and more portable when it creates that bundle under `tmp_path` instead of pinning to one historical sample directory under `data/golden/benchmark-vs-golden/`.

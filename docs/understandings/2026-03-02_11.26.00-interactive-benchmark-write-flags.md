## Discovery

Date: 2026-03-02

- The interactive benchmark paths in `cookimport/cli.py` (`single_offline` and
  single-profile all-matched) were previously hardcoding
  `write_markdown=True` and `write_label_studio_tasks=True` when building
  benchmark kwargs, so Markdown + Label Studio task sidecars were always generated.
- The two interactive benchmark entrypoints now resolve write toggles from
  environment defaults, defaulting both to `False`:
  - `COOKIMPORT_BENCH_WRITE_MARKDOWN`
  - `COOKIMPORT_BENCH_WRITE_LABELSTUDIO_TASKS`
- `cookimport/c3imp_entrypoint.py` already seeds these env vars to `0`, so
  default `C3imp` sessions keep these writes disabled unless users explicitly
  set env vars to `1` before launch.

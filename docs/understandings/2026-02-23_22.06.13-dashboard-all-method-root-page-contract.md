# Dashboard all-method root-page contract

- Prior behavior only wrote all-method pages when grouped runs existed, so `all-method` pages were absent on dashboards with zero grouped runs.
- Current behavior always writes `data/output/.history/dashboard/all-method-benchmark.html` as part of the main static site.
- Grouped-run detail pages now live beside `index.html` (for example `all-method-benchmark__<run_timestamp>__<source_slug>.html`) rather than under a separate subfolder.
- Grouping/ranking logic is unchanged and remains CSV-first using artifact paths containing `all-method-benchmark/<source_slug>/config_*`.

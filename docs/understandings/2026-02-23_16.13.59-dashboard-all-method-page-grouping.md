# All-method dashboard grouping discovery

- All-method sweeps already append one benchmark CSV row per configuration via the normal `labelstudio-benchmark` path; no extra metrics store is required.
- The stable grouping key for standalone pages is the benchmark artifact path segment pattern: `.../all-method-benchmark/<source_slug>/config_*`.
- Dashboard rendering can stay CSV-first by grouping existing `benchmark_records[*].artifact_dir` (or `run_dir`) and ranking configs from the persisted benchmark metrics (`precision`, `recall`, `f1`, `practical_f1`, `recipes`, run-config fields).
- Standalone page filenames are derived from `<run_timestamp_folder>__<source_slug>.html`; dashboard now writes a shared in-site root page at `all-method-benchmark.html` with detail pages as sibling files.

# Discovery: all-method all-golden matching rules

- Interactive all-method now supports `All golden sets with matching input files` and resolves source hints in strict order:
  1. run `manifest.json` `source_file`,
  2. first non-empty `freeform_span_labels.jsonl` row `source_file`,
  3. first non-empty sibling `freeform_segment_manifest.jsonl` row `source_file`.
- Matching is by exact filename (`Path.name`) against top-level importable files in `data/input`.
- Segment-manifest fallback is required because some freeform exports have empty/metadata-only `freeform_span_labels.jsonl`; without that fallback, valid gold exports were skipped despite available source metadata.

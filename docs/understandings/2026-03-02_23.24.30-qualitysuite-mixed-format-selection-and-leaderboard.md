---
summary: "QualitySuite mixed-format support now comes from suite-time extension coverage plus optional leaderboard extension slices."
read_when:
  - "When changing quality suite target selection across EPUB/PDF mixes"
  - "When adding or debugging quality leaderboard by-format outputs"
---

- Discovery selection is now extension-aware at suite build time: with capped `--max-targets`, it seeds one target per extension first (when capacity permits) before strata fill.
- `source_extension` is now persisted per target so downstream tools can avoid re-deriving format from paths.
- `bench quality-run` summary/report now expose `format_counts` and `selected_format_counts`, making PDF starvation visible before compare/leaderboard.
- `bench quality-leaderboard --by-source-extension` computes separate rankings inside each extension bucket while keeping the global leaderboard unchanged.

# Format Naming Conventions

- `RecipeSage` is an importer/source format label, not the pipeline's canonical intermediate format name.
- Intermediate outputs (`intermediate drafts/*.jsonld`) should be described as `schema.org Recipe JSON`.
- Final outputs (`final drafts/*.json`) should be described as `cookbook3`.
- `RecipeDraftV1` may still appear as an internal model/schema identifier, but user-facing docs/help text should prefer `cookbook3`.

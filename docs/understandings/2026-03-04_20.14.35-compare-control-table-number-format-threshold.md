Discovery:
- Group-summary table values are rendered in one categorical branch of `renderCompareControlPanel`, so a single helper function can enforce consistent formatting across `Rows`, `Avg`, and secondary metric columns.
- Threshold-based formatting (`abs(value) > 5`) keeps large token-like values readable while preserving precision for ratio-style metrics near zero.
- `toLocaleString("en-US")` is the right seam for comma insertion in dashboard JS output.

Code refs:
- `cookimport/analytics/dashboard_render.py`: `compareControlTableNumber(...)` and categorical group-table rendering calls.
- `tests/analytics/test_stats_dashboard.py`: JS assertions for helper/threshold/commas behavior.

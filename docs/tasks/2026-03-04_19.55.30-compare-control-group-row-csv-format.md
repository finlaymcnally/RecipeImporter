## Problem
Compare & Control `Group outcome means` rows were formatted as `Group: avg X (N rows) | stat | stat`, which made spreadsheet paste awkward and hid row count away from the group name.

## Acceptance Criteria
- Row format is `Group (N rows): avg X, Stat A Y, Stat B Z`.
- Secondary stats are comma-separated, not pipe-separated.
- Row count appears beside the group label.
- Dashboard JS test coverage asserts the new format shape.

## Verification
- `. .venv/bin/activate && pytest tests/analytics/test_stats_dashboard.py -k dashboard_js_supports_compare_control_analysis`

## Evidence
- Test result: `1 passed, 76 deselected in 0.54s`.

## Constraints / Decision
- Updated only the single row-string seam in `renderCompareControlPanel` to avoid changing compare/control math or filtering behavior.
- Added one focused assertion in existing dashboard JS coverage test.

## Rollback
- Revert the dashboard string-format hunk in `cookimport/analytics/dashboard_render.py` and the paired assertion/docs entries.

## Problem statement

`bench speed-run --include-codex-farm` could be triggered without an explicit user approval gate. We need SpeedSuite to keep Codex Farm support available while requiring positive confirmation first.

## Acceptance criteria

- `bench speed-run` fails fast when `--include-codex-farm` is set without an explicit confirmation value.
- `bench speed-run` accepts Codex Farm only when the explicit confirmation token is provided.
- `run_speed_suite(...)` enforces the same confirmation contract for direct Python callers.
- Speed/CLI docs mention the required confirmation argument.

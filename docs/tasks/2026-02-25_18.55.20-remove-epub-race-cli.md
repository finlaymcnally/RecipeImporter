# Remove EPUB race CLI feature

## Problem statement

Interactive menu and direct `cookimport epub` commands still exposed EPUB race mode, but this workflow should be removed from the product surface.

## Acceptance criteria

- Interactive main menu does not include `EPUB debug: race extractors on one file`.
- `cookimport epub race` is not a registered subcommand.
- Race-only backend modules are removed (`cookimport/parsing/epub_auto_select.py`, `cookimport/parsing/extraction_quality.py`).
- Race compatibility fields are removed from runtime/report/analytics contracts (`epubAutoSelection`, `epubAutoSelectedScore`, `epub_auto_selected_score`).
- Docs and conventions no longer describe race as a current workflow.

## Verification steps

- `source .venv/bin/activate && pytest tests/cli/test_c3imp_interactive_menu.py tests/cli/test_epub_debug_cli.py`
- `source .venv/bin/activate && pytest tests/cli/test_c3imp_interactive_menu.py tests/cli/test_epub_debug_cli.py tests/labelstudio/test_labelstudio_benchmark_helpers.py -k "epub_race or interactive_main_menu_does_not_offer_inspect"`
- `source .venv/bin/activate && pytest tests/analytics/test_perf_report.py tests/analytics/test_stats_dashboard.py`
- `source .venv/bin/activate && pytest tests/analytics/test_perf_report.py tests/analytics/test_stats_dashboard.py tests/cli/test_c3imp_interactive_menu.py tests/cli/test_epub_debug_cli.py tests/cli/test_cli_output_structure.py tests/labelstudio/test_labelstudio_benchmark_helpers.py`

## Evidence

- `14 passed, 2 warnings` for the focused CLI test run.
- `3 passed, 94 deselected, 2 warnings` for the targeted regression selection.
- `60 passed, 2 warnings` for focused analytics+CLI regression run (`test_perf_report`, `test_stats_dashboard`, interactive menu + EPUB debug CLI tests).
- `149 passed, 7 warnings` for combined analytics + CLI + labelstudio regression run after full compatibility-field removal.

## Constraints and decision

- Kept EPUB debug commands `inspect|dump|unpack|blocks|candidates|validate`.
- Removed race wiring/commands and deleted race-only backend scorer modules.
- Removed remaining race compatibility fields from reports/manifests/history/dashboard contracts.

## Rollback plan

- Restore `_interactive_epub_race(...)` + `epub_race` menu branch in `cookimport/cli.py`.
- Restore `@epub_app.command("race")` in `cookimport/epubdebug/cli.py`.
- Revert related test/docs changes in one commit.

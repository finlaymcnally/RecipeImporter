# Tasks

Use this folder for task-spec files that define scope, acceptance criteria, verification, and evidence for a change.

Suggested filename format:
- `YYYY-MM-DD_HH.MM.SS - short-title.md`

Stage-specific deep notes can still live in stage folders (for example `docs/05-staging/`), but task contracts referenced by commit messages should live here.

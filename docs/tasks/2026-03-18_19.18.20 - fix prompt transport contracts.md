## Problem statement

Two active prompt contracts are obviously wrong for agent execution.
The knowledge prompt still talks like a file-backed task while the runtime sends inline JSON, and the active file-backed line-role wrapper lost most of the task semantics that exist in the richer inline prompt.

## Acceptance criteria

- Knowledge direct prompts no longer contain `{{INPUT_PATH}}` or file-reading instructions.
- Knowledge direct prompts explicitly describe the inline JSON transport.
- Active file-backed line-role prompts keep the compact tuple contract but also include the minimum semantic guidance needed to classify rows reliably.
- Focused tests lock both contracts.

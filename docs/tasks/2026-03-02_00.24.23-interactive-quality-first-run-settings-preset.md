# Task: Interactive quality-first run-settings preset

## Problem statement

Interactive mode had global/preferred/quality-suite-winner/last/edit options, but no always-available built-in option for the known quality-first parser winner stack.

## Acceptance criteria

- Interactive run-settings chooser exposes a new option for the quality-first winner stack.
- Selecting that option applies:
  - `epub_extractor=unstructured`
  - `epub_unstructured_html_parser_version=v1`
  - `epub_unstructured_preprocess_mode=semantic_v1`
  - `epub_unstructured_skip_headers_footers=true`
- Existing chooser options continue to work unchanged.
- Tests cover option presence and applied settings.
- CLI docs mention the new chooser option.

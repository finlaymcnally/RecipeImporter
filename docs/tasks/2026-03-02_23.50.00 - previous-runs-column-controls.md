## Problem

`Previous Runs` used fixed hard-coded columns. Users could not reorder columns, resize widths, or add/remove fields for ad-hoc comparisons.

## Acceptance Criteria

- `Previous Runs` includes UI controls to add/remove fields dynamically.
- Users can reorder visible columns in the dashboard UI.
- Users can resize column widths from table header drag handles.
- Existing all-method grouping and rules filtering behavior still render correctly with dynamic columns.
- Analytics dashboard renderer tests cover the new HTML/JS/CSS contract.

## Problem statement

`bench quality-run --include-codex-farm` could be triggered without an explicit user approval gate. We need QualitySuite to keep Codex Farm support available while requiring positive confirmation first.

## Acceptance criteria

- `bench quality-run` fails fast when `--include-codex-farm` is set without an explicit confirmation value.
- `bench quality-run` accepts Codex Farm only when the explicit confirmation token is provided.
- `run_quality_suite(...)` enforces the same confirmation contract for direct Python callers.
- Bench docs mention the required confirmation argument.

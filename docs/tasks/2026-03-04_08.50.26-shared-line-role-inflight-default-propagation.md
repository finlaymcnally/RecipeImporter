## Problem statement
Canonical line-role codex inflight behavior was wired in specific CLI flows, so new processing/benchmark paths could miss the same defaults.

## Acceptance criteria
- Shared prediction-generation seam applies defaults automatically: non-split jobs use `8`, split-gated jobs use `4`.
- Explicit `COOKIMPORT_LINE_ROLE_CODEX_MAX_INFLIGHT` env override still takes precedence.
- Interactive benchmark wrappers no longer need dedicated inflight env wiring.
- Tests cover parsing override behavior and ingest single-vs-split defaults.

## Verification steps
- `source .venv/bin/activate && pytest -q tests/parsing/test_canonical_line_roles.py -k "codex_progress_callback_reports_batch_counts or codex_max_inflight_override"`
- `source .venv/bin/activate && pytest -q tests/labelstudio/test_labelstudio_ingest_parallel.py -k "line_role_projection_updates_draft_fields or line_role_uses_split_gated_inflight_default"`
- `source .venv/bin/activate && pytest -q tests/labelstudio/test_labelstudio_benchmark_helpers.py -k "interactive_single_offline_codex_enabled_runs_vanilla_then_codex_and_writes_comparison or interactive_single_offline_codex_disabled_runs_only_vanilla_and_skips_comparison"`
- `source .venv/bin/activate && pytest -q tests/labelstudio/test_labelstudio_benchmark_helpers_single_profile.py -k "interactive_single_profile_parallel_requests_two_live_spinner_slots or interactive_single_profile_all_matched_codex_runs_vanilla_then_codex_per_book"`

## Evidence
- Added `codex_max_inflight` optional override to `label_atomic_lines(...)` and precedence test.
- Added ingest-level resolver tied to split-phase gating and env-override precedence.
- Updated `generate_pred_run_artifacts(...)` to pass resolved inflight override into `label_atomic_lines(...)`.
- Removed interactive-only CLI inflight wrappers.

## Constraints / decision
- Kept parser default and env behavior intact; shared defaults are injected by ingest at runtime so new call paths inherit policy without per-flow wiring.

## Rollback / compensation
- Revert ingest seam override wiring and restore CLI-specific wrappers if needed.

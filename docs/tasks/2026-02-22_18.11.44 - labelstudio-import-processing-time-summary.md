# Problem statement

Label Studio import summaries showed project/task/prelabel details but not total runtime. We need the terminal summary to include processing time for easier run auditing.

# Acceptance criteria

- Interactive Label Studio import summary prints `Processing time: ...`.
- Non-interactive `cookimport labelstudio-import` summary prints `Processing time: ...`.
- Duration formatting is human-readable (`Xs`, `Ym Zs`, `Xh Ym Zs`).
- A focused CLI test covers the new summary line.
- Docs that describe this summary output are updated.

# Constraints and decisions

- Reused existing CLI import flow and measured elapsed time around `_run_labelstudio_import_with_status` to avoid touching ingest internals.
- Added one shared formatter helper to keep interactive and non-interactive output consistent.

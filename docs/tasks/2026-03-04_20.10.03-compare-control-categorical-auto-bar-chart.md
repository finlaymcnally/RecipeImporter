## Problem statement

Compare & Control dynamic chart always rendered as scatter. For numeric outcome + categorical compare fields, this is harder to read than bars.

## Acceptance criteria

- Numeric compare fields still render scatter points.
- Categorical compare fields automatically render bar series.
- Split-by still creates separate series.
- Saved valid Compare & Control selections auto-render chart on load/state-restore (no manual re-toggle required).
- Categorical bar points render with per-group soft pastel colors (not one heavy single-color fill).
- Categorical bar styling uses lower visual weight (lighter borders/spacing/title styling).
- Categorical group colors stay stable when local subset filtering changes visible category count.
- Categorical group colors stay stable when category frequencies/order change in filtered row mixes.
- Existing compare/control chart harness tests pass with updated expectations.

## Verification steps

1. `source .venv/bin/activate && pytest tests/analytics/test_stats_dashboard.py -k "compare_control_dynamic_chart_builds_scatter_from_visible_rows or dashboard_js_supports_compare_control_analysis"`

## Evidence

- `pytest ... -k "compare_control and not pixel and not drift"` -> `6 passed, 73 deselected`

## Constraints / gotchas

- Dashboard JS is generated from `cookimport/analytics/dashboard_render.py` string templates.
- Keep compare/control activation gate behavior unchanged.

## Decision

Kept the existing builder seam and added chart-type auto resolution: numeric compare fields route to scatter builder, categorical compare fields route to new bar builder. Also auto-activate chart rendering when a valid non-discover compare selection is already present in restored state. Bar series now color each group point from a calm pastel palette and use lighter chart styling to reduce visual weight. Color mapping is now deterministic from compare-field + group-key hash (not visible subset order or frequency rank), so per-group colors persist across subset and row-mix changes.

## Rollback / compensation plan

Revert this task diff to return to scatter-only compare/control chart rendering.

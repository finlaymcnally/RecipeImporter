# Task: Local editable codex-farm recipe pass prompts

## Problem statement

Make recipe codex-farm stage prompts editable as plain text files under `llm_pipelines/prompts`, mirroring the existing freeform prelabel template workflow.

## Acceptance criteria

- `llm_pipelines/pipelines/` contains pass pipeline specs for:
  - `recipe.chunking.v1`
  - `recipe.schemaorg.v1`
  - `recipe.final.v1`
- Each pipeline spec references a prompt file under `llm_pipelines/prompts/`.
- Prompt text can be edited directly in `.prompt.txt` files.
- `llm_pipelines/schemas/` contains output schemas for pass1/pass2/pass3 aligned with `cookimport/llm/codex_farm_contracts.py`.
- A test verifies default pipeline files, referenced prompt files, and referenced schema files are present.

## Verification steps

Run from repo root:

    source .venv/bin/activate
    pip install -e .[dev]
    pytest -q tests/test_llm_pipeline_pack.py tests/test_codex_farm_orchestrator.py

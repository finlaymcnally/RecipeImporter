## Problem statement

`docs/plans/speed.md` had core matcher-selector code but missing proof-depth requirements: opcode parity coverage, byte-identical end-to-end alignment output checks, and completed plan evidence/progress updates.

## Acceptance criteria

- Canonical eval emits a deterministic alignment artifact that can be compared byte-for-byte across matcher modes.
- Parity tests cover opcode and matching-block equivalence across stress-case text pairs.
- End-to-end canonical test verifies stdlib vs auto mode produce identical alignment output bytes.
- Speed plan is updated with completed progress checkboxes and real evidence snippets.
- Optional local micro-benchmark script exists for quick stdlib-vs-auto timing checks.

## Verification steps

From repo root:

1. `source .venv/bin/activate`
2. `PIP_CACHE_DIR=/tmp/.pip-cache python -m pip install -e '.[benchaccel]'`
3. `pytest tests/bench/test_sequence_matcher_dropin_parity.py tests/bench/test_eval_stage_blocks.py -q`
4. `python scripts/bench_sequence_matcher_impl.py --tokens 1400 --repeats 3`
5. Run the documented A/B canonical eval snippet from `docs/plans/speed.md` Artifacts and Notes.

## Evidence

- `pytest` targeted suites exited `0`.
- Micro-benchmark script showed opcode parity and faster auto-mode timings:
  - `stdlib best_seconds=6.078813`
  - `auto impl=cydifflib best_seconds=0.738722`
  - `opcode_parity=True`
- A/B canonical eval snippet reported:
  - `score_equal=True`
  - `alignment_equal=True`
  - `aligned_bytes_equal=True`
  - `stdlib_impl=stdlib`
  - `auto_impl=cydifflib`

## Constraints and decision

- Constraint: no algorithmic change allowed for canonical alignment.
- Decision: add a deterministic output artifact (`aligned_prediction_blocks.jsonl`) and strengthen tests around that artifact rather than changing alignment logic.
- Constraint: local environment had pip cache permissions under `$HOME`.
- Decision: use `PIP_CACHE_DIR=/tmp/.pip-cache` for `.[benchaccel]` install in verification commands.

- Set `COOKIMPORT_BENCHMARK_SEQUENCE_MATCHER=stdlib` to force baseline runtime behavior.

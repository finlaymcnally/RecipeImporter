# Problem

The shared workspace-worker watchdog already tolerates bounded inline Python heredocs, but the knowledge worker launch path still overrode that policy and forced `watchdog_inline_python_heredoc_forbidden` kills.

# Acceptance Criteria

- Main knowledge workspace-worker launches do not pass `forbid_inline_python_heredocs=True`.
- Shared watchdog coverage includes the ExecPlan command shapes: execution-root `cd`, local `cp`/`mv`, and bounded heredocs.
- The explicit callback seam can still forbid inline Python heredocs when a caller opts in on purpose.

# Constraints / Decision

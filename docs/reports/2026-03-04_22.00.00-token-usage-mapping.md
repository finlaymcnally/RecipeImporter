## High-Token Rows That Are Not Evidence Of Hidden Vanilla
The earlier version got this wrong.

The large history rows on March 2-3 are real, but they are overwhelmingly the intended `/codexfarm` runs. For example:

- `2026-03-03_12.12.49` `/codexfarm`: `19,688,398`
- `2026-03-03_01.24.28` `/codexfarm`: `15,249,983`
- `2026-03-02_23.33.49` `/codexfarm`: `19,052,717`
- `2026-03-02_23.15.01` `/codexfarm`: `13,665,192`

The matching official single-offline `/vanilla` rows are not the source of those totals.

The CLI path for official single-offline vanilla explicitly forces:

- `llm_recipe_pipeline=off`
- `line_role_pipeline=off`
- `atomic_block_splitter=off`

So those rows should not be described as secret hidden-AI vanilla examples.

## 3. Root Cause Attribution
1. **Config Hash Fragmentation:** Variations in non-LLM settings (e.g. `epub_extractor`) invalidated the LLM cache, forcing redundant calls for identical text blocks.
2. **Invisible Defaults:** `line_role_pipeline` defaulting to `codex-line-role-v1` allowed LLM calls during runs intended to be "deterministic" or "vanilla."
3. **Weak CLI Gating:** The `--include-codex-farm` flag failed to gate the `line_role_pipeline`.

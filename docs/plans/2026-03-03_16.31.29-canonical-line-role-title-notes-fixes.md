## Surprises & Discoveries

- Observation: Deterministic logic can emit `RECIPE_TITLE` for obvious all-caps titles, but canonical candidate allowlists can omit `RECIPE_TITLE`, making codex correction impossible.
  Evidence: Local reproduction in prior investigation showed title-like lines receiving only `OTHER`/`KNOWLEDGE` candidates in common paths.

- Observation: Note prose is frequently marked `instruction_like` by deterministic sentence heuristics before codex can help.
  Evidence: Run-level error analysis showed many wrong note lines with `decided_by=rule` and instruction predictions.

- Observation: Yield boundary regex matched bare `serving` inside prose and split note lines into false yield tails.
  Evidence: New note regression tests initially split `... before serving ...` into two atomic lines, with the second line tagged `yield_prefix`.

## Context and Orientation

Relevant files:

- `cookimport/parsing/recipe_block_atomizer.py` provides deterministic atomization features and candidate-label hints.
- `cookimport/parsing/canonical_line_roles.py` builds candidate allowlists, applies deterministic labels, and decides codex escalation.
- `llm_pipelines/prompts/canonical-line-role-v1.prompt.md` defines codex instructions/examples for line roles.
- `tests/parsing/test_recipe_block_atomizer.py` and `tests/parsing/test_canonical_line_roles.py` cover deterministic and canonical behavior.

Key failure mode: codex cannot output labels outside candidate allowlists. If allowlists omit `RECIPE_TITLE`, or deterministic titles are escalated into a codex path that cannot return title, titles collapse to non-title labels.

## Plan of Work

Patch `recipe_block_atomizer.py` to keep `RECIPE_TITLE` available in reasonable recipe and near-recipe contexts, especially when deterministic title cues are present even outside strict recipe spans.

Patch `canonical_line_roles.py` to avoid escalating low-confidence deterministic `RECIPE_TITLE` predictions when escalation would remove title from the feasible label set. Keep codex escalation for truly ambiguous cases.

Refine deterministic instruction-likeness heuristics to reduce false positives for note-like prose (`RECIPE_NOTES`) without weakening genuine instruction detection. Add targeted tests for notes and title behavior.

Optionally add a concise prompt example if test evidence shows codex confusion remains after gating/heuristic fixes.

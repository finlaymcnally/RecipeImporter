# Benchmark extractor wiring

Finding:
- Benchmark prediction generation (`labelstudio-benchmark` and offline pred-run helpers) ultimately uses `generate_pred_run_artifacts(...)`.
- That path records and applies `epub_extractor` from `C3IMP_EPUB_EXTRACTOR`.

Implication:
- If benchmark command paths do not set `C3IMP_EPUB_EXTRACTOR`, extractor choice falls back to environment/default behavior.
- Exposing a benchmark-level extractor choice is best implemented by setting this env var for the prediction-import call scope.

# Label Studio export project root (2026-02-15)

What changed:
- Default export destination is now:
  - `<output_dir>/<project_slug>/exports/...`
- This applies to interactive and non-interactive `labelstudio-export` runs when `--run-dir` is not provided.

What stayed the same:
- Manifest lookup by `project_name` still resolves `project_id` and enforces task-scope alignment.
- `--run-dir` still forces export output into a specific run directory.

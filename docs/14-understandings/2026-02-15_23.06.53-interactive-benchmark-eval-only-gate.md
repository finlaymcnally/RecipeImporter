# Interactive benchmark eval-only gate

Finding:
- In `_interactive_mode`, `labelstudio_benchmark` defaults to `upload` and only asks `How would you like to benchmark?` when both `gold_candidates` and `prediction_runs` are non-empty.
- Candidate discovery scans `data/golden` and `data/output` for `**/exports/freeform_span_labels.jsonl` plus `**/label_studio_tasks.jsonl` (where the parent folder is treated as the prediction run).

Implication:
- Upload mode runs directly when either artifact set is missing, which means at least one required discovery set was not found locally.

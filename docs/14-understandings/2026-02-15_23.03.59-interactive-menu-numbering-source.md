# Interactive menu numbering source

Finding:
- `_interactive_mode()` builds per-flow choices, but every select-style prompt routes through `_menu_select()` in `cookimport/cli.py`.
- `_menu_select()` is therefore the single control point for option numbering, shortcut-key enablement, and Backspace-to-go-back behavior.

Implication:
- For consistent UX, new interactive menus should use `_menu_select()` instead of calling `questionary.select()` directly.

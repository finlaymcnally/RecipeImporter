# Label Studio export timestamped project root (2026-02-15)

Superseded:
- Current behavior changed at `2026-02-15_22.45.08`; export now writes to `data/golden/<project_slug>/exports/...` by default.
- See `docs/14-understandings/2026-02-15_22.45.08-labelstudio-export-project-root.md`.

What changed:
- `run_labelstudio_export(...)` no longer uses the latest manifest parent as `run_root` when `--run-dir` is not supplied.
- Default export destination is now always:
  - `<output_dir>/<timestamp>/labelstudio/<project_slug>/exports/...`

What stayed the same:
- Manifest lookup by `project_name` still resolves `project_id` and enforces `task_scope` alignment.
- `--run-dir` still forces export output into the specified run directory.

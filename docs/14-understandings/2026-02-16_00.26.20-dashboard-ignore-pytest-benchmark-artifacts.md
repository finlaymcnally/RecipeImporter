# Discovery

Recent benchmark rows were being sourced from benchmark CSV entries whose artifact paths pointed to pytest temp folders (for example `pytest-46/test_labelstudio_benchmark_pas0/eval`).

# Rule

Dashboard benchmark collection should skip benchmark artifacts matching pytest temp eval/prediction layout (`.../pytest-<n>/test_*/eval` or `.../pytest-<n>/test_*/prediction-run`) so local Python tests do not pollute user-facing benchmark history.
